# 7 属性克制设计稿

## 目标
7 属性版用于 MVP 到下一阶段之间的扩展，不追求覆盖所有幻想题材属性，而是先验证：**属性是否能驱动训练师提取形态、出场阵容和走位判断。**

设计原则：
- 克制倍率保持 2 倍，抵抗倍率保持 0.5 倍。
- 每个属性至少有一个明确的战术身份。
- 克制关系尽量符合直觉，但允许为战棋节奏服务。
- 暂不做免疫。免疫会让 20x20 小战斗里出现“完全没事做”的回合，MVP 先避免。
- 未来双属性按防御属性逐项连乘，因此同时被克制会受到 4 倍伤害。

## 属性列表
### 火
定位：爆发、清场、压制草和冰。

战术意义：
- 适合打脆弱或怕火的敌人。
- 容易被水和地面压制，不能无脑前压。

### 水
定位：稳定、防守、修复、反制火和地。

战术意义：
- 适合训练师安全形态和续航路线。
- 被草和雷针对，不能长期停留在水形态。

### 草
定位：控制、缠绕、阵地战、反制水和地。

战术意义：
- 适合削稳定度、拖慢大怪、保护训练师。
- 怕火、冰、飞，站位需要更谨慎。

### 冰
定位：限制移动、惩罚草和飞。

战术意义：
- 适合打高机动敌人或控制敌人路线。
- 被火和地压制，属于高收益但脆的属性。

### 雷
定位：速度、打断、反制水和飞。

战术意义：
- 适合配合 CTB 行动轴，未来可做加速、延迟、麻痹类效果。
- 被地面压制，遇到地属性敌人需要换形态或换队友处理。

### 飞
定位：机动、绕后、反制草和地。

战术意义：
- 适合高移动单位、越过地形威胁、打后排。
- 被雷和冰针对，不能只靠机动性无风险压场。

### 地
定位：厚重、防守、反制火和雷。

战术意义：
- 适合作为 Boss、大怪、盾型单位属性。
- 被水、草、冰、飞压制，弱点多但抗压和伤害稳定。

## 克制表
攻击方克制防御方时造成 2 倍伤害。

| 攻击属性 | 克制 |
|----------|------|
| 火 | 草、冰 |
| 水 | 火、地 |
| 草 | 水、地 |
| 冰 | 草、飞、地 |
| 雷 | 水、飞 |
| 飞 | 草、地 |
| 地 | 火、雷、冰 |

## 抵抗表
抵抗不是额外单独设计，而是克制关系的反向：如果 A 克制 B，那么 B 攻击 A 时造成 0.5 倍伤害。

例子：
- 火克制草，所以草打火为 0.5 倍。
- 雷克制水，所以水打雷为 0.5 倍。
- 地克制雷，所以雷打地为 0.5 倍。

## 当前 MVP 落地方式
第一阶段只把属性表扩到 7 属性，不急着给每个属性都做特殊状态。

已落地：
- `Enums.ElementType` 已扩展 `ICE`、`ELECTRIC`、`FLYING`、`GROUND`。
- `TypeChart.gd` 已扩展 7 属性 `MATCHUPS`。
- 电花鼠已改为雷属性，新增冰羽兽作为冰属性后备。
- 样板战新增飞羽小怪和地壳小怪，用来验证雷/冰/地/飞的克制关系。
- 伤害预览继续显示倍率，例如 `克制x2`、`克制x4`、`抵抗x0.5`。

暂不做：
- 麻痹、冰冻、飞行无视地形、地面免疫雷等复杂规则。
- 完整 18 属性。
- 双属性单位大量出现。

## MVP 样板战调整建议
为了不让战斗一下子变复杂，样板战里先只加入少量新属性：

- 电花鼠已改为雷属性后备，替代“电花鼠按无属性结算”的临时设定。
- 水针小怪保留水属性，用雷形态处理会更清晰。
- 厚血大怪保持草属性，继续让火狐兽和火系技能有爆发爽点。
- 已新增地壳小怪，用来压制雷形态，避免玩家一直用雷形态。
- 已新增飞羽小怪，用来压制草形态，并给雷/冰形态提供明确目标。

这样第一场战斗能验证：
- 火打草 Boss 的爆发爽感。
- 水形态安全但怕草。
- 雷形态能处理水，但会被地属性反制。
- 草形态能控场，但怕火/冰/飞。

## 后续扩展方向
如果 7 属性版成立，再考虑 18 属性。18 属性不建议照搬任何现有作品的完整表，而是围绕 elomon 的系统重做：
- 哪些属性服务 CTB 行动轴。
- 哪些属性服务地形。
- 哪些属性服务捕捉和稳定度。
- 哪些属性服务训练师提取形态。

属性数量越多，越需要 UI 和教学支撑。否则玩家记不住克制关系，属性就会从策略变成查表。
